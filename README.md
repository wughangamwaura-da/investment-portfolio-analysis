# Investment Portfolio Analysis Using SQL, Python and Dashboard Visualization

## Project Overview

This project analyzes an investment portfolio using SQL, Python, and dashboard visualization tools to evaluate portfolio performance, risk exposure, allocation, and profitability. The workflow involved:

* Data cleaning and preprocessing
* SQL querying and aggregation
* Python statistical analysis
* Dashboard visualization and reporting

The project integrated SQL for structured data analysis, Python for financial calculations and risk analysis, and Looker Studio dashboards for interactive reporting and visualization. Financial analytics objectives included portfolio return analysis, volatility measurement, Sharpe Ratio evaluation, asset allocation analysis, and performance ranking.

# Tools Used

* Python
* SQL
* pandas
* matplotlib
* Looker Studio
* Google Sheets

# Key Analyses

## Portfolio Return Analysis

Calculated asset and portfolio returns to identify profitable and underperforming investments.

## Risk Analysis

Measured portfolio volatility using standard deviation to evaluate investment risk exposure.

## Sharpe Ratio

Evaluated risk-adjusted portfolio performance using the Sharpe Ratio.

## Asset Allocation

Analyzed investment distribution across multiple asset categories to assess diversification and concentration risk.

## Performance Ranking

Ranked assets based on returns, allocation, and risk-adjusted performance.
